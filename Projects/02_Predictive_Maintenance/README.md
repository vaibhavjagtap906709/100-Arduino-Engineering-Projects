# Project 02: Industrial Predictive Maintenance System using Vibration Analysis

## Category
Industrial Automation IoT

## Difficulty
Advanced

## GitHub Repository
arduino-industrial-predictive-maintenance-vibration

## Problem Statement
Unexpected industrial machine failures cost manufacturers millions annually. Traditional maintenance is either reactive (post-failure) or scheduled (wasteful). Vibration-based predictive maintenance can detect bearing failures 2-4 weeks before they occur.

## Objectives
- Monitor vibration FFT spectrum, temperature, and current of industrial motors
- Detect anomalies indicating bearing wear, imbalance, or misalignment
- Predict remaining useful life (RUL) using threshold comparison
- Alert maintenance teams via industrial HMI and email/SMS

## Key Components
- Arduino Due
- MPU6050 6-DOF IMU Module
- DS18B20 Waterproof Temp Probes
- ACS712 30A Current Sensor
- OLED 128x64 I2C Display
- ESP32 Dev Board
- Active Buzzer Module
- RGB LED Module
